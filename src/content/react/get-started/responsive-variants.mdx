            ## Responsive Variants

            Tailus UI React, built on top of [Tailus Themer](/themer/get-started/introduction), incorporates [Tailwind Variants](https://www.tailwind-variants.org/docs/getting-started#responsive-variants-optional). This means you can use responsive variants directly within Tailus UI React.

            To use responsive variants, provide an object to the prop. In this object, the key should represent the breakpoint, and the corresponding value should represent the desired prop value at that breakpoint.
